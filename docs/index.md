## Especificaciones

### Robot

| Parámetro | Valor |
|-----------|-------|
| Radio de rueda | 0.075 m |
| Wheelbase | 0.180 m |
| Track width | 0.250 m |
| Encoders | 4096 CPR |

### Hardware

- **Computadora**: NVIDIA Jetson Orin
- **Motores**: RoboClaw 2x15A  (4 motores)
- **Sensores**: PX4 (IMU/GPS), RPLiDAR
- **Control**: PS4 Controller, FrSky FR6
